Данный проект выполнен в рамках задания в Школе21. Краткие описания задач содержатся в этом файле.

## Саммари заданий

### Задание 1. Знакомство с инструментами
Развертывание n8n на облачной платформе Cloud.ru с использованием Docker и Container Apps. Настройка подключения к GigaChat API и создание первого воркфлоу, демонстрирующего ответ LLM на запрос.

### Задание 2. Твой первый ИИ-агент. Выбор специализации
Создание Telegram-бота, который задаёт пользователю серию вопросов (опыт, интересы, цели, данные из датасета «Школы 21») и на основе ответов выдаёт рекомендацию по IT-специализации. Реализация через n8n с использованием AI Agent, SimpleMemory и GigaChat 2.

### Задание 3. Дайджест новостей
Настройка автоматического парсинга постов из 3+ Telegram-каналов по выбранной специализации (через RSS-прослойку). AI-агент суммаризирует посты и раз в день присылает дайджест в приватный Telegram-канал.

### Задание 4. Анализ вакансий
Раз в неделю парсинг свежих вакансий с hh.ru через API. AI-агент выделяет топ-15 наиболее частых навыков и требований, приводит их к нормализованному виду (например, python/питон → python) и отправляет отчёт в Telegram.

### Задание 5. Персональный ментор
Создание бота, который уточняет тему для изучения, формирует контент-план (темы → понятия) и последовательно присылает посты с объяснениями. Используется Static Data для отслеживания пройденных тем и Tavily Search для поиска актуальных материалов.

## Полученные и применённые навыки

- Развёртывание и настройка n8n в облачной среде (Cloud.ru, Docker, Container Apps)
- Интеграция Telegram-ботов с n8n (Webhook, триггеры, отправка сообщений)
- Работа с LLM через API (GigaChat 2, настройка промптов, system message)
- Создание AI-агентов с памятью (SimpleMemory, Static Data)
- Парсинг Telegram-каналов через RSS и вакансий через API hh.ru
- Автоматическая суммаризация текста и извлечение сущностей (навыков)
- Нормализация данных (приведение навыков к единому формату)
- Интеграция поисковых инструментов (Tavily Search)
- Организация периодических задач (рассылки раз в день/неделю)
- Управление контекстом диалога и состоянием пользователя
- Написание системных промптов для сложного поведения агента
